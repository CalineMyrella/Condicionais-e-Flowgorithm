# Condicionais e Algoritmos com Flowgorithm e VisuAlg

Este repositório contém uma série de exercícios práticos focados no desenvolvimento de lógica de programação, utilizando estruturas condicionais, operadores aritméticos e relacionais. As atividades foram desenvolvidas como parte do curso de **Desenvolvimento de Sistemas no SENAI**.

##  Tecnologias Utilizadas

* **Flowgorithm:** Ferramenta para criação e execução de fluxogramas.
* **VisuAlg:** Interpretador de pseudocódigo (Portugol).
* **Git/GitHub:** Para versionamento e organização dos arquivos.

##  Estrutura do Repositório

### Exercícios Inclusos:

| Atividade | Descrição |
| :--- | :--- |
| **Idade** | Verifica se uma pessoa é maior de idade ou possui restrições. |
| **Soma de Números** | Recebe valores e realiza o processamento aritmético simples. |
| **Comissão do Vendedor** | Calcula o valor da comissão com base no volume de vendas e percentuais pré-definidos. |
| **Média de Notas** | Calcula a média escolar e define se o aluno está aprovado ou reprovado. |
| **Sistema de Senha** | Validação simples de acesso através de comparação de strings. |
| **Volume do Paralelepípedo** | Cálculo matemático de volume (Comprimento × Largura × Altura). |
| **Voto** | Verifica a obrigatoriedade ou possibilidade de voto com base na idade. |
